## The Layout Problem

Representation of a graph.

**By Sets**:

- Graph: {Node, Edge}
- Node: {v1, v2, ..., vn}
- Edge: {e1, e2, ..., em}

Node = {v1, v2, v3, v4}
Edge = {{v1, v2}, {v2, v3}, {v1, v4}}

**By adjacency list**:

v1: v2, v8
v2: v1, v3
v3: v2, v5, v9, v10
...

The layout problem:

(1) The position of nodes -- `Node -> (Number, Number)`
(2) The curve of edges -- `Edge -> Interval -> (Number, Number)`
